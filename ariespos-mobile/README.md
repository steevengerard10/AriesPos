# ARIESPos Mobile

App móvil en React Native (Expo) que replica la interfaz y funciones del POS de escritorio:
- Login
- Venta y cobro
- Búsqueda y selección de productos
- Carrito
- Selección de cliente
- Fiado
- Descuentos
- Observaciones
- Alertas en tiempo real (Socket.IO) para ventas canceladas y productos eliminados
- Escaneo de códigos de barra con la cámara

> Esta app está pensada para funcionar igual que la versión host/escritorio, pero adaptada a móvil.
